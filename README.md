# Weather App (Android)

A simple weather application built using Android Studio that displays real-time weather information.

## Features
- Real-time weather updates
- Splash screen and login screen
- Weather history tracking
- Clean UI using XML layouts

## Tech Stack
- Java
- Android Studio
- XML (UI Design)

## Project Structure
- app/src/main/java → Main app logic
- app/src/main/res → UI layouts and resources
- AndroidManifest.xml → App configuration

## How to Run
- Open project in Android Studio
- Sync Gradle
- Run on emulator or device
